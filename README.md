# ERP-Pele-de-Seda
Modelagem de banco de dados relacional (ERP) para a clínica Pele de Seda. Centraliza a gestão de múltiplas filiais, escalas de profissionais volantes e o ciclo de atendimentos. Destaque para o controle de estoque híbrido: gerencia tanto as vendas diretas ao cliente quanto a baixa fracionada de insumos consumidos nos procedimentos estéticos.
